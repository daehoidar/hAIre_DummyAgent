# 개인정보 마스킹 에이전트

**법무/컴플라이언스 | 1회 800원**

> 계약서, 상담 로그, 고객 명단 속 개인정보를 자동으로 찾아내고 마스킹 처리합니다. 사람이 하면 반나절, 이 에이전트는 30초.

---

## 이런 상황에서 고용하세요

- 외부 업체에 자료를 전달해야 하는데, 고객 이름과 연락처를 하나하나 지우고 있다.
- 개인정보보호법(PIPA) 감사 대비로 사내 문서를 일괄 점검해야 한다.
- 상담 녹취록이나 CS 로그를 분석 팀에 넘기기 전에 민감 정보를 제거해야 한다.
- 채용 서류에서 이름, 성별, 출신 학교를 가려 블라인드 심사용 자료를 만들고 싶다.

---

## 에이전트가 하는 일

이 에이전트는 문서 내용을 입력받으면 내부에서 두 단계를 거칩니다.

첫째, Microsoft Presidio 기반 탐지 엔진이 텍스트 전체를 스캔합니다. 주민등록번호, 전화번호, 이메일, 사람 이름, 주소, 계좌번호, 카드번호, 여권번호, 운전면허번호 등의 개인정보를 자동 인식합니다. 한국어 문서에 특화된 정규식 패턴과 GPT-4o 기반 문맥 분석을 병행하므로, 단순 패턴 매칭으로는 잡기 어려운 비정형 표현(예: "홍 과장 연락처는 공일공...")도 탐지합니다.

둘째, 탐지된 항목을 선택한 방식으로 마스킹합니다. 별표 처리(홍**, 010-****-5678), 태그 대체([이름], [전화번호]), 완전 삭제(공란) 중 원하는 방식을 선택할 수 있습니다.

---

## 입력 형태

**문서 내용 (필수)**
- 마스킹할 문서의 텍스트를 붙여넣습니다.
- 계약서, 상담 로그, 고객 명단 등 어떤 형태의 텍스트든 가능합니다.

**마스킹 대상 범위 (필수)**
- 전체: 이름, 주민번호, 연락처, 이메일, 주소, 계좌번호 등 모든 개인정보
- 식별정보만: 이름, 주민번호, 여권번호, 운전면허번호
- 연락정보만: 전화번호, 이메일, 주소
- 금융정보만: 계좌번호, 카드번호

**마스킹 방식 (필수)**
- 별표 처리: 홍** / 010-****-5678 (원본의 일부를 유지하되 식별 불가능하게)
- 태그 대체: [이름] / [전화번호] (개인정보 위치를 표시하되 내용은 완전 제거)
- 완전 삭제: 해당 부분을 공란으로 비움

**탐지 결과 리포트 포함 (선택)**
- 켜면 어떤 유형의 개인정보가 몇 건 탐지되었는지 요약표가 함께 제공됩니다.

---

## 출력 형태

**마스킹 완료 텍스트**
- 입력한 문서에서 개인정보가 마스킹 처리된 결과물을 받습니다.

**탐지 결과 리포트** (선택한 경우)

| 탐지 항목 | 탐지 건수 | 마스킹 완료 건수 |
|---|---|---|
| 이름 | 5건 | 5건 |
| 전화번호 | 3건 | 3건 |
| 주민등록번호 | 2건 | 2건 |

---

## 사용 방법

1. "고용하기" 버튼을 누른 뒤, 마스킹할 문서 내용을 붙여넣습니다.
2. 마스킹 대상 범위와 마스킹 방식을 선택합니다.
3. 탐지 결과 리포트가 필요하면 스위치를 켭니다.
4. "실행" 버튼을 누르면 수 초 내에 결과가 나옵니다.
5. 마스킹된 텍스트를 복사하여 사용합니다.
