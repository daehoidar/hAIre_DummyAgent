# 트렌드 스카우트 & 카피라이터

**마케팅 | 1회 900원**

> 키워드 하나 입력하면 최신 트렌드 분석부터 인스타그램 게시물 대본까지, 마케팅팀의 일주일치 콘텐츠가 완성됩니다.

---

## 이런 상황에서 고용하세요

- 매일 아침 네이버 트렌드, 구글 트렌드, 경쟁사 인스타를 돌며 2시간씩 조사하고 있다.
- "요즘 뭐가 뜨는지" 감은 있는데, 그걸 우리 브랜드 말투로 콘텐츠화하는 게 오래 걸린다.
- 숏폼 대본, 카드뉴스 대본, 블로그 포스팅을 동시에 만들어야 하는데 손이 부족하다.
- 경쟁사가 어떤 콘텐츠로 반응을 얻고 있는지 체계적으로 파악하고 싶다.

---

## 에이전트가 하는 일

이 에이전트는 세 단계를 거쳐 마케팅 콘텐츠를 생산합니다.

첫째, 트렌드 조사입니다. 입력한 키워드의 검색량 추이를 웹 검색으로 분석합니다. 연관 키워드, 급상승 키워드, 계절적 패턴을 파악합니다. 경쟁사 SNS URL을 입력한 경우, 해당 계정의 최근 인기 게시물 패턴(소재, 형식, 발행 시간대)도 분석합니다.

둘째, 콘텐츠 기획입니다. 트렌드 분석 결과를 바탕으로, 선택한 유형(카드뉴스, 숏폼 대본, 블로그, 인스타 캡션, 또는 전체)의 콘텐츠를 기획합니다. 제목, 훅(첫 문장), 핵심 메시지를 먼저 설계한 뒤 본문을 작성합니다.

셋째, 브랜드 톤 적용입니다. 입력한 톤앤매너 설명에 맞추어 전체 콘텐츠의 말투, 이모지 사용 빈도, 문장 길이를 조정합니다. 해시태그 추천과 발행 캘린더 제안까지 포함됩니다.

---

## 입력 형태

**분석할 키워드 (필수)**
- 트렌드를 조사할 주제를 입력합니다.
- 예: 비건 뷰티, 제로 웨이스트, MZ세대 건강식품

**경쟁사 SNS URL (선택)**
- 인스타그램, 네이버 블로그 등 경쟁사 채널 주소를 줄바꿈으로 구분하여 입력합니다.

**브랜드 톤앤매너 설명 (필수)**
- 우리 브랜드가 사용하는 말투, 타겟 연령대, 분위기를 자유롭게 설명합니다.
- 예: "20대 여성 타겟, 밝고 친근한 말투, 이모지 적극 사용, ~해요체"

**주요 콘텐츠 유형 (필수)**
- 카드뉴스 대본 (5~7장 분량)
- 숏폼 대본 (30~60초, 릴스/숏츠용)
- 블로그 포스팅 초안
- 인스타그램 캡션 5개
- 전체 (위 4종 모두 생성)

**세트 수 (필수)**
- 3세트 (일주일치 콘텐츠) 또는 5세트 (2주치 콘텐츠)

---

## 출력 형태

**트렌드 분석 리포트**
- 키워드 검색량 추이, 연관 키워드, 경쟁사 인기 콘텐츠 패턴

**콘텐츠 대본 세트**
- 선택한 유형별로 완성된 대본이 세트 수만큼 제공됩니다.
- 각 대본에는 제목/훅, 본문, 해시태그 추천이 포함됩니다.

**발행 캘린더 제안**

| 발행일 | 콘텐츠 유형 | 제목 | 핵심 메시지 |
|---|---|---|---|
| 월요일 | 카드뉴스 | 비건 뷰티 입문 가이드 | 성분표 읽는 법부터 시작 |
| 수요일 | 릴스 대본 | 비건 립 3종 발색 비교 | 직접 발라봤더니 이게 1등 |
| 금요일 | 인스타 캡션 | 주말 셀프케어 루틴 | 비건 뷰티로 나를 위한 시간 |

---

## 사용 방법

1. "고용하기" 버튼을 누른 뒤, 분석할 키워드를 입력합니다.
2. 경쟁사 SNS URL이 있으면 추가합니다. (선택)
3. 브랜드 톤앤매너를 설명합니다.
4. 콘텐츠 유형과 세트 수를 선택합니다.
5. "실행" 버튼을 누르면 1~3분 내에 결과가 나옵니다.
