# 썸네일 & 배너 디자이너

**디자인/크리에이터 | 1회 400원**

> 콘텐츠 주제와 스타일만 알려주면, 유튜브 썸네일이나 블로그 배너 시안 여러 개가 바로 생성됩니다.

---

## 이런 상황에서 고용하세요

- 영상은 편집 끝났는데 썸네일 만들 시간이 없다.
- 캔바(Canva)로 배너를 만들어 보는데, 디자인 감각이 부족해서 결과가 아쉽다.
- 매주 콘텐츠를 올리는데, 매번 새 썸네일 디자인하는 게 반복 작업이다.
- 인스타그램 피드 통일감을 맞추고 싶은데, 일관된 스타일로 이미지를 만들기 어렵다.

---

## 에이전트가 하는 일

이 에이전트는 두 단계를 거쳐 이미지를 생성합니다.

첫째, 이미지 생성입니다. 콘텐츠 주제와 선택한 스타일을 바탕으로 GPT-4o가 최적의 이미지 프롬프트를 설계합니다. 이 프롬프트를 DALL-E 3에 전달하여 고품질 이미지를 생성합니다.

둘째, 텍스트 합성 및 사이즈 조정입니다. 이미지 위에 지정한 텍스트(제목, 에피소드 번호 등)를 합성합니다. 선택한 용도에 맞는 정확한 해상도(유튜브 1280x720, 인스타 1080x1080 등)로 최종 이미지를 출력합니다.

---

## 입력 형태

**콘텐츠 주제/제목 (필수)**
- 이미지에 담고 싶은 내용을 자유롭게 적습니다.
- 예: "자취생 냉장고 파먹기 레시피", "코딩 입문자를 위한 Python 강의"

**이미지 용도 (필수)**
- 유튜브 썸네일 (1280x720)
- 블로그 대표 이미지 (1200x630)
- 인스타그램 피드 (1080x1080)
- 인스타그램 스토리 (1080x1920)

**디자인 스타일 (필수)**
- 미니멀/깔끔, 굵은 텍스트/임팩트, 따뜻한/감성적, 레트로/빈티지, 테크/미래적, 팝/컬러풀 중 선택

**이미지 위 텍스트 (선택)**
- 썸네일에 넣을 제목이나 강조 텍스트. 예: "5분 완성!", "TOP 10"

**선호 색상 (선택)**
- 원하는 색상 계열이나 hex 코드. 예: "파란색 계열", "#FF5733"

**시안 개수 (필수)**
- 3개 또는 5개 중 선택

---

## 출력 형태

**생성된 시안**
- 선택한 개수만큼 PNG 형식의 이미지가 생성됩니다.
- 각 이미지는 지정한 해상도에 정확히 맞추어 출력됩니다.
- 텍스트 오버레이가 적용된 최종 버전으로 바로 업로드에 사용할 수 있습니다.

**시안별 디자인 의도 설명**
- 각 시안이 어떤 의도로 디자인되었는지 간단히 설명됩니다.
- 예: "시안 1: 음식 클로즈업으로 식욕을 자극하는 구도, 노란색 텍스트로 시선 집중"

---

## 사용 방법

1. "고용하기" 버튼을 누른 뒤, 콘텐츠 주제를 입력합니다.
2. 이미지 용도와 디자인 스타일을 선택합니다.
3. 텍스트, 색상을 입력합니다. (선택)
4. 시안 개수를 선택하고 "실행" 버튼을 누릅니다.
5. 30초~1분 내에 시안이 생성됩니다.
